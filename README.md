## ğŸš€ How It Works

1. The webcam captures live video input
2. **MediaPipe Hands** detects hand landmarks
3. Gesture patterns are analyzed in real time
4. Recognized gestures are converted into text
5. Text is spoken aloud using **Text-to-Speech**

All processing happens **locally in the browser**.
